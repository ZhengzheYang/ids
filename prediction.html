Task Description

Given the dataset, we create a vectorized sparse matrix from the review texts to predict the corresponding rating for that business within 5 classes. Since the matrix will be huge, we adopt mini-batch fitting technique to process the dataset batch-by-batch.

The following classifers are used:

  • Stochastic Gradient Descent (SGD)
  • Perceptron
  • Multinomial Naive Bayesian
  • Passive-Aggressive

First, the results given by the vectorized model is reported. Detailed training time and improvement by batches are also presented. Lastly, the feature importances are extracted and plotted using simple Logistic Regressions on a few of unigram features.

Results

      The results are evaluated using simple accuracy metric. As we can see, the Stochastic Gradient Descent and Passive-Aggressive models perform the best, with an accuracy above 58%+.

      This plot shows each model's accuracy with the number of training examples. We can see that all models fit approximately at the same pace, and the reason might be the size of our data is relatively small. Also, Perceptron and Passive Aggressive model achieve the best performance.

      This plot shows each model's accuracy through trianing time, we can also draw the conclusion that all models converge at approximately the same pace.

      This is plot shows the overall fitting time of each model, and with this plot we can see that the Perceptron actually converge fastest while Naive Bayesian Multinomial takes the most time to converge.

     We use logistic regression to predict review stars with features: 'review_count', 'is_open', 'avg_star', 'useful', 'funny', and 'cool'. The prediction accuracy on test data of logistic regression is 45.03%. We visualize the feature importance with heatmap for each feature and we can see that 'funny', 'useful' and 'avg_star' are actually of vital importance to the prediction result. For example, if the avg_star of a business is low the review rating should be low correspondingly. Also, if many people think a review to be funny then this review tend to be of predicted as of high rating, on the other hand, if most people think a review to be useful then this review is more likely to be negative.
